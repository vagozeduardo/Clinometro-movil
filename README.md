# Clinómetro Móvil
**Estado:** Una versión base funcional, En **Desarrollo** para mejoras.


---

## Descripción General
Desarrollo de una aplicación móvil para ayudar a realizar los cálculos para
determinar la altura de un objeto mediante el uso de un teléfono celular.
Aplicación nativa para dispositivos **Android** desarrollada en **Java y Android Studio**.
La idea primciapl es lograr medir la **altura** y la **inclinación angular** de objetos o estructuras distantes (ej. edificios, árboles) usando cámara y sensores del dispositivo.

Este proyecto demuestra mi habilidad para:
- Aplicar fórmulas trigonométricas en código Java.
- Diseñar una interfaz limpia y funcional en Android.
